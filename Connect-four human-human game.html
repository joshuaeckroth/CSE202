Connect-four human-human game

Task

Write a program that allows two humans to play Connect-four. The game is briefly described below.

Connect-four

Although I'm sure there are variants on this game, what's described here is how we will play it.

The game board is a 6x7 grid (6 rows, 7 columns). There are two players. The players' turns alternate, and during each turn, a player places a token in one of the columns. The token slides to the "bottom" of the column, as if the game board was vertical (6 rows vertically, 7 rows horizontally). So the first move of the game would have the token at the bottom of the column in which it is place (the token would be at row 6); if the other player then plays in the same column, that player's token would be above the other player's token (so the new token is at row 5), etc.

A player wins when four tokens, from the same player, are arranged either horizontally (all on the same row), vertically (all in the same column), or diagonally (each token one column left or right, one row up or down, from another token), consecutively (no breaks).

Call one player "A" and the other "B". Label their respective tokens "A" and "B" also. Then an empty game board looks like this (with the columns labeled with numbers):

Group work

You can work in groups of 2 or 3 if you'd like. I need to know who you worked with, so write all your names in a comment at the top of your code. You only need to submit one copy of the code from one person's email address. You will all receive the same grade.

Next steps

After you complete this program, you'll be asked to make player "A" or player "B" a computer player, so you will have one or more extra functions in your code that look at the game grid and determine the best move (or the worst move, depending). Rather than asking the user for a move from player "A" (or "B"), your program will just make the move automatically.

Then we will have a quick competition in class, in which I link your computer player with others that the class has programmed, and my own. We'll see whose player is statistically best (by running something like 1000 games for each pairing). The winning group, or myself, gets a couple bars of vegan chocolate (sorry if you don't want chocolate).
