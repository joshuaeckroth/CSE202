Assignment 3

Task

Write a program that repeatedly asks for positive integers (values greater than or equal to 1). The user may type 0 to quit the program. The user will enter at least one value before typing 0 to quit. After the user types 0 to quit, your program should report how often the largest number was entered.

Examples

Enter positive integers (0 to quit):
1
0
Largest input: 1. Number of 1's: 1.

Enter positive integers (0 to quit): 5 5
3 0
Largest input: 5. Number of 5's: 2.

Enter positive integers (0 to quit):
5
5
8
5
0
Largest input: 8. Number of 8's: 1.

Enter positive integers (0 to quit): 5 5 3 5 5 0
Largest input: 5. Number of 5's: 4.

My solution

#include <iostream>
using namespace std;
int main()
{
    int n, max = 0, maxcount = 0;
    cout << "Enter a positive integer (0 to quit): ";
    cin >> n;

    while(n != 0)
    {
        if(n > max)
        {
            max = n;
            maxcount = 1;
        }
        else if(n == max)
        {
            maxcount++;
        }

        cout << "Enter a positive integer (0 to quit): ";
        cin >> n;
    }

    cout << "Done. Largest input: " << max << ". Number of "
         << max << "'s: " << maxcount << "." << endl;

}
