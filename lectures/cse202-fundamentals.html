About C++:

  • one of many languages
  • a superset of C -- C code still passes as C++ code
  • an ALGOL-like structure and syntax
    • C++ has lots of braces, semicolons
    • Each command is usually typed on a new line, runs top to bottom, left to right
    • Doesn't have to be that way: don't really need semicolons and braces (Python), don't really need top-down left-right (Haskell), don't really need to type in files (Max)
  • We are learning C++ for a host of reasons: historical (was C or Pascal previously, which are similar), practical (common today), social (languages pushed by tool makers), etc.
  • Overall point is to utilize computers in our research and jobs
    • This can be done with virtually any programming language
    • Topics in this class, could be taught with virtually any language
      • It is also relatively hard to teach C++ because of all the syntax and other oddities (mostly relating to the fact it's compatible with the older C language)
