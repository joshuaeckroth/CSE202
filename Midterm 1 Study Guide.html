Midterm 1 Study Guide

The first midterm is 48 minutes, in class, on paper. So it is a short test.

Make sure you know how to work out the following:

  • what variable "types" (double, int, bool, char, string) are appropriate for specific situations
  • the results of basic math involving +, -, *, / (both integer "quotient" and double "division"), and % (integer "remainder")
  • the results of basic boolean operations like ! (not), && (and), and || (or), as well as comparisons like <, >, <=, ==, !=, etc.
  • for if and else if and else situations, "what values of the variables in the conditionals will cause certain parts of the code to be visited or not visited?"
  • for a while() loop, "how many times is the conditional checked?"
  • for a while() loop, "what is the value of each variable after the loop completes?"
  • for a while() loop, "what values of the variables in the conditional will cause the loop to stop?"
  • for any situation, "what does this code print?"
